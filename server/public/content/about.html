CODE IN THE DARK

Il Code In The Dark è il primo contest in Italia dedicato al coding “alla cieca”, un contest front-end (HTML, CSS) in cui ogni concorrente gareggia per implementare il design di un sito web dato solo uno screenshot come punto di partenza. È il pubblico a decretare il vincitore, il King of the code.

ORGANIZZATO DA

Nasciamo dall’underground hacker, siamo cresciuti diventando un sistema di aziende basato sulla cultura Agile, e conserviamo gelosamente una filosofia molto aperta ed un livello di skill tecnologico elevato. Realizziamo qualsiasi tipo di piattaforma informatica per i più svariati settori. Non vendiamo prodotti, ma progetti. Ci piacciono le cose complicate e amiamo organizzare eventi che riflettano il nostro modo di “connettere le passioni”.
